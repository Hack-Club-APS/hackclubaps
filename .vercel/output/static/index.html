What happens at Hack Club?

We meet weekly to build, learn, and share.

Workshops

Guided sessions where we learn new languages, frameworks, and tools. From Python to React to Game Dev.

Hardware

We don't just do software. We mess with Arduinos, soldering, and building physical gadgets.

Hackathons

We participate in global Hack Club events and host our own internal project jams.

We ship projects.

Hack Club isn't a class. It's a club. We don't have tests or grades. We have projects. Whether it's a personal portfolio, a discord bot, or a video game, you'll build something you're proud of.

  • Build your own website
  • Create a game in JS
  • Design a PCB
  • Deploy a web app
